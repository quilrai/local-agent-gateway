Usage Guide

Getting Started

  1. Download and launch the Agent Gateway app
  2. Click on "How to Use"
  3. Select your coding agent (Claude Code, Codex, or Cursor)
  4. Click Install for automated setup, or follow the instructions for manual setup

Once setup is complete, the dashboard and message logs will be populated as you start sending messages.

Adding Custom Detection Patterns

The gateway includes pre-defined patterns for common sensitive data (API keys, credentials, etc.). You can add your own:

  1. Open the Detections tab in the app
  2. Click Add Pattern
  3. Choose pattern type:
    • Block - Reject requests containing this pattern
    • Redact - Replace matched content before forwarding
  4. Enter your regex pattern or simple text match
  5. Test the pattern against sample input
  6. Save and enable the pattern

Setting Rate Limits

Prevent runaway requests and control costs with rate limiting:

  1. Open the Settings tab
  2. Navigate to Rate Limiting
  3. Configure limits:
    • Token warning threshold - Get notified when requests exceed this
    • Token hard limit - Block requests above this count
    • Requests per minute - Throttle request frequency
  4. Choose action: Warn or Block
  5. Save settings

Custom LLM Gateway

Use your own API tokens with any OpenAI-compatible endpoint:

  1. Open the Settings tab
  2. Navigate to Custom Endpoints
  3. Click Add Endpoint
  4. Configure:
    • Endpoint URL - Your LLM provider's API URL
    • API Key - Your token (stored locally)
    • Model mapping - Optional model name overrides
  5. Enable monitoring for this endpoint
  6. All requests to this endpoint will now flow through the gateway
